# Roadmap

## Fase 0 - Base limpia

- Monorepo inicial.
- UI clínica base.
- API con pacientes, ficha clínica y auditoría.
- Migración inicial.
- OpenAPI exportable.
- Gobierno para agentes IA.

## Fase 1 - Ficha clínica operacional

- Autenticación y roles.
- CRUD completo de alergias, medicación, signos vitales y documentos.
- Búsqueda clínica segura.
- Panel longitudinal por paciente.
- Tests de integración con PostgreSQL.

## Fase 2 - IA local controlada

- Proveedor Ollama local.
- Prompts versionados.
- Evaluaciones clínicas y de seguridad.
- Trazabilidad de sugerencias IA.
- Controles para revisión humana obligatoria.

## Fase 3 - Producción sanitaria

- Cifrado de datos sensibles.
- Backups y recuperación.
- Observabilidad sin PHI.
- Revisión legal y de cumplimiento.
- Auditoría inmutable o append-only reforzada.
