# Auditoria Incremental

Esta carpeta contiene reportes reproducibles de bajo ruido. Los reportes informativos no bloquean CI; los checks bloqueantes deben tener baseline explicito y fallar solo ante regresiones nuevas.

Comandos:

- `npm run audit:variables-domain-snapshots`
- `npm run audit:permissions-read-access`
- `npm run check:paper-traceability`
- `npm run check:architecture`
