# EPIS2 Lessons Imported

EPIS2 queda como cantera de aprendizaje, no como base runtime de OneEpis.

## Importado como concepto

- ficha como memoria clinica activa
- humano confirma antes de escritura final
- timeline y evolucion S/O/A/P
- auditoria de outputs IA
- modo papel serio

## No importado

- Dashboard sobredimensionado.
- Command center como pantalla principal.
- Triple modo visual.
- Rutas legacy.
- Pantallas completas.
- Gates historicos pesados como condicion inicial.
- Dependencias visuales de EPIS2.

## Regla

Si una idea de EPIS2 entra en OneEpis, entra como principio o contrato reducido, no como copia de modulo.

Las reglas activas viven en `docs/GOVERNANCE.md`, `docs/AI_CHART_CORE.md` y
`docs/PROGRESSIVE_DEVELOPMENT_PLAN.md`.
